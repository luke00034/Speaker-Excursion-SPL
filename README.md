### 揚聲器怎麼壞掉的?

常見揚聲器的發聲原理主要透過把振動傳到空氣讓你聽到音樂，而這個振動系統的極限也常常拿來類比揚聲器的極限。
* 我工作上較常見的損壞大概有以下幾種也順便做個回顧
> 1. 音圈無法再運動軌道內運動(卡在軌道上、脫離軌道... ... )
> 2. 振模破掉了(振破掉、甩破掉、戳破掉XD)
> 3. 喇叭燒了(斷線、冒煙了... ...是還沒有看過起火的啦~)
> 
 對應上述狀況整理了幾個方式可以掌握他是不是快壞掉了(到極限了)。
 
 有的! 上班閒暇中剛好有整理了一些評估方法，看看怎麼樣會壞掉了。 下面來聊聊~ 
#### 音圈無法在運動軌道內運動

關於這一點我有整理了一些工作上的小技巧，通常音圈有一定的運動軌道我們可以從軌道的幾何資訊知道喇叭合理的運動範圍是多少，接下來就能透過預測揚聲器的運動軌跡去知道怎麼去設定合理的工作範圍(輸入功率)給揚聲器了
> EX.喇叭就只能在上下5公分運動，那你就想辦法不要讓它超過阿~幹!
> 預測方式如下:
> 透過量測得到喇叭參數，然後帶入運動方程式~ 搞定🤝！ 很簡單吧!
> 計算機算一算、喇叭測一測~恩  可以下班了
